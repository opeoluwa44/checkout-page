## Overview

![screenshot](https://user-images.githubusercontent.com/16707738/92399059-5716eb00-f132-11ea-8b14-bcacdc8ec97b.png)

This Checkout page is fully responsive across all devices, with a little bit of javascript functionality 
which allows for item count and automatic update of the total price and shipping cost at checkout point:

-Users can select the number of items to checkout 
- Users can see an automatic display of the total cost of items selected.
- Users can also see an updated shipping cost of the items
- users can fill the checkout form and submit successfully.
